## ğŸ“ Customization

### Update Personal Information

1. **Contact Details** - Edit `src/components/Contact.jsx`
2. **Projects** - Modify the projects array in `src/components/Projects.jsx`
3. **Skills** - Update skill categories in `src/components/Skills.jsx`
4. **About Section** - Change content in `src/components/About.jsx`
5. **Social Links** - Update links in `src/components/Footer.jsx`

### Styling

All styles are modular and component-specific:
- Each component has its own CSS file
- Global styles are in `src/index.css`
- App-level styles are in `src/App.css`
